# Teste DQRTech

## Install Steps:


```bash
$ git clone git@github.com:fernandohcorrea/teste-dqrtech.git
```

```bash
$ cd teste-dqrtech/
```

```bash
$ npm i
```

```bash
$ npm run build
```

```bash
$ ./cli 
```

```bash
$ ./cli install
```

```bash
$ docker compose up --build
```
